# Assets

This folder contains the app's static assets.

## Required Files

### App Icons
- `icon.png` - Main app icon (1024x1024 px)
- `adaptive-icon.png` - Android adaptive icon foreground (1024x1024 px)
- `favicon.png` - Web favicon (48x48 px)
- `splash-icon.png` - Splash screen icon (300x300 px)

### Fonts
Place the following fonts in the `fonts/` directory:
- `SpaceMono-Regular.ttf` - Regular weight
- `SpaceMono-Bold.ttf` - Bold weight
- `IBMPlexMono-Regular.ttf` - Monospace regular
- `IBMPlexMono-Bold.ttf` - Monospace bold

You can download these from:
- Space Mono: https://fonts.google.com/specimen/Space+Mono
- IBM Plex Mono: https://fonts.google.com/specimen/IBM+Plex+Mono

## Color Scheme

The app uses a neo-brutalist design with:
- Primary: #FF0000 (Red)
- Accent: #FFD700 (Gold)
- Background: #FFFFFF (Light) / #121212 (Dark)
- Border: #000000

## Icon Design Guidelines

Icons should follow the neo-brutalist aesthetic:
- Bold geometric shapes
- Thick black borders (2-3px stroke)
- High contrast colors
- No rounded corners (or very minimal)
- Raw, intentional visual weight
